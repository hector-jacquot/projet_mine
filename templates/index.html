{{ APP_NAME }}

Démonstrateur IoT pour la gestion de la sécurité et des facteurs environnementaux dans une mine (projet d'ingénierie ISEP).

  • Capteurs : température, humidité, luminosité, présence, CO₂, CH₄.
  • Actionneurs : buzzer ON dès qu'un seuil critique est franchi.
  • Dashboard : dernier relevé, statut OK/Alerte, historique via Chart.js.
  • Administration : modification dynamique des seuils en base.

Démarrer

Créez un compte, connectez-vous, puis consultez le dashboard.

Connexion Inscription

Architecture

Flask (routes + API JSON), MySQL (pymysql), templates Jinja2, Bootstrap et Chart.js.

Compatibilité Drizzle

Tables et colonnes en snake_case et clés simples pour rester compatibles avec Drizzle ORM côté TypeScript.
